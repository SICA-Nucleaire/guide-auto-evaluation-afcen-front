Test de connaissance au code RCC-E



Contexte

L’application a pour objectif d’aider un fournisseur à diagnostiquer sa propre maitrise technique du code RCC‑E mise en œuvre dans le cadre d’une commande de matériel. L’application devra permettre au fournisseur d’identifier et de comprendre les points d’amélioration qu’il doit travailler. Ce diagnostic ne se substitue pas à l’évaluation du code par un audit du code complet et ne préjuge pas des résultats de cet audit.



Référentiel

Il est pris comme terminologie de référence le code RCC-E Edition 2019. Ce questionnaire peut être utilisé sur d’autres versions avec les limites suivantes :

  • Les chapitres indiqués dans le questionnaire peuvent ne pas correspondre à la version utilisée du RCC-E,
  • Certains thèmes développés dans le questionnaire peuvent ne pas être présent dans la version utilisée du RCC-E.

Après avoir renseigné l'interface et cliqué sur "Valider", si le nombre de thème sélectionner dépasse 30 minutes de tests, l'interface vous proposera le choix suivant :
Souhaitez-vous faire :

  • Un test complet avec une durée estimée de xx min
  • Un test partiel par tirage aléatoire des thèmes sélectionnés afin de rester dans la durée estimée de 30 minutes.
